**Find recipes based on the ingredients you already have at home.**  
CookWithWhat helps you discover meals using whatâ€™s in your kitchen, shows missing ingredients, and allows filtering by category.
This project is built with **React + Vite** and uses the [MealDB API](https://www.themealdb.com/) to fetch recipes and meal data.

## Features

- Search for recipes using your available ingredients.
- See which ingredients you are missing for each recipe.
- Filter recipes by category.
- Responsive design for desktop and mobile.
- Embedded YouTube video tutorials for recipes.
